# Silence Detection

Absence is a primary data type. Silence is not nothing — it is measurable, classifiable, and in many cases more significant than noise.

## Silence Types

- **ORGANIC** — Natural absence. Source is down, topic is dormant, nothing to report.
- **COORDINATED** — Multiple sources simultaneously stop covering something they were previously covering. Clean boundaries. High political alignment coverage.
- **UNKNOWN** — Insufficient data to classify.

## Silence Fingerprint

Three measurable dimensions:

### Boundary Clarity (0.0 to 1.0)
- 0.0 = ragged, organic — coverage trails off naturally
- 1.0 = clean, coordinated — coverage stops simultaneously across sources
- Clean boundaries are a coordination signal

### Political Alignment Coverage (0.0 to 1.0)
- How uniformly the silence spans across politically diverse sources
- 1.0 = universal silence across all political alignments
- Universal silence is a HIGH severity signal

### Displacement Events
- A loud story breaking simultaneously with a silence event
- If displacement confirmed → alert severity elevated
- The loud story may be the mechanism of the silence

## Escalation

An event that generates universal media silence but spikes the GCP network is the strongest possible signal of coordinated narrative suppression.

Silence persisting through N recursive investigation rounds → ESCALATE.
