SECTION 1 — WHAT THE GCP IS
================================================================================

The Global Consciousness Project is an international collaboration operating
since 1998 out of Princeton University's Engineering Anomalies Research Lab,
now under the Institute of Noetic Sciences.

The system maintains a network of 65+ hardware Random Event Generators (REGs)
— also called eggs — distributed across host sites worldwide. Each egg
generates a continuous stream of random data, recording 200-bit trials once
per second. All data is synchronized and transmitted to a central server in
Princeton, creating a real-time global dataset of parallel random sequences.

The hypothesis: when major world events generate widespread simultaneous
emotional or attentional response across large populations, the output of the
REG network shows statistically significant deviation from expected randomness.

After testing 500 formally defined events across nearly 20 years of operation,
the cumulative result departs from chance expectation at odds exceeding a
trillion to one. The nature of this effect remains under investigation. The
data itself is not disputed.

The GCP is fully open access under GPL license. Raw data and processing
algorithms are available for free download. A real-time feed is available
at gcpdot.com, updated every minute, running 10 minutes behind live generation.
A Node.js API wrapper exists that returns the current GCP value as hex.
